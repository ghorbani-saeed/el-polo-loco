# El-Polo-Loco 🎮

**El-Polo-Loco** ist ein unterhaltsames 2D-Action-Spiel, bei dem der Spieler als Huhn durch verschiedene Levels läuft, Hindernisse überwindet, Münzen sammelt und gegen Gegner kämpft. Das Spiel enthält Animationen, Soundeffekte, Hintergrundmusik und einfache Levelmechaniken.  

---

## 🛠 Technologien

- HTML  
- CSS  
- JavaScript  

---

## 🎮 Features

- Bewegungen des Charakters mit der Tastatur  
- Sammeln von Münzen  
- Verschiedene Gegner und Endbosse  
- Soundeffekte und Hintergrundmusik  
- Start- und Game Over-Bildschirme
